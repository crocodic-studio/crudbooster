# Toggle Form Type

### Code Sample
```php
$this->form[] = ['label'=>'Enabled','name'=>'is_enabled','type'=>'toggle', 'value'=>'1'];
```

### Legend
| Attribute Name | Example |
| -------------- | ------- |
| value | Use `1` to enable by default or `0` to disable |
