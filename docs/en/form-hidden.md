# Hidden Form Type

### Code Sample
```php
$this->form[] = ['label'=>'Label Name','name'=>'field_name','type'=>'hidden','value'=>'YOUR_CUSTOM_VALUE'];
```
By put `value` attribute, it will force the value to your setted value
